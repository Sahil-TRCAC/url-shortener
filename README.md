# ⚡ ShortLink Pro - URL Shortener with Analytics

A powerful URL shortener web application with built-in analytics, Google OAuth authentication, and real-time click tracking.

## 🚀 Features

- **URL Shortening** - Convert long URLs into short, manageable links
- **Click Analytics** - Track total clicks, device types, and click history
- **Google OAuth** - Secure authentication using Google accounts
- **Admin Panel** - Traditional login for admin users
- **Responsive Design** - Beautiful UI that works on all devices
- **Real-time Dashboard** - Live statistics and analytics
- **Copy to Clipboard** - Easy link copying functionality
- **Device Detection** - Track whether clicks come from mobile, tablet, or desktop

## 🛠️ Tech Stack

- **Backend:** Django 4.2, Django REST Framework
- **Authentication:** Google OAuth 2.0
- **Database:** SQLite (development) / PostgreSQL (production)
- **Frontend:** HTML5, CSS3, JavaScript
- **Deployment:** Ready for Render, PythonAnywhere, or Heroku

## 📋 Prerequisites

- Python 3.11 or higher
- pip (Python package manager)
- Google Cloud account (for OAuth credentials)
